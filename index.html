About:

This form takes the email address and converts it into the hex code for each character. It then creates a mailto: tag composed of these codes.

This page will output the HTML mailto: tag containing those codes which you will need to copy and put in your page where you want you address to appear.

Someone using a normal browser will see your address and be able to mail you via a standard mailto: link, however any e-mail harvesting agents will merely see something like %74%72%65%65%74%6D%65%40%68%6F%74%6D%61%69%6C%2E%63%6F%6D instead of a real email address.

Of course, someone could write an email harvester that could convert this all back to your email address. However the likelihood of that is remote.
